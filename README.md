# D.A.D. ComfyUI Workshop Landing Page

Landing page for **AI-Assisted Architectural Visualisation using ComfyUI** workshop by Drupad Ashar Designs (D.A.D.)

## Workshop Details

- **Target Audience:** Architecture students at I.D.E.A.S, Nagpur
- **Duration:** 3 sessions × 4 hours each (12 hours total)
- **Modules:** 9 modules covering floor plans, exteriors, interiors, materials, and style transfer
- **Registration Fee:** ₹1,750 per participant
- **Date:** May 2026 (exact date TBC)

## Features

- 🎨 **Modern Design:** Dark theme with D.A.D. brand colors (Obsidian, Marigold, Parchment)
- 📱 **Mobile Responsive:** Fully optimized for all device sizes
- 🖱️ **Interactive Elements:** Expandable module details with process chain navigation
- 🔗 **ComfyUI Node UI:** Section headers styled as ComfyUI node interfaces
- ⚡ **Smooth Animations:** JS-drawn bezier wire connections and transitions
- 📝 **Registration Form:** Direct submission to FormSubmit

## Technical Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Fonts:** Fraunces (display), DM Sans (body)
- **Forms:** FormSubmit integration
- **Icons:** Unicode symbols and custom SVG graphics
- **Responsive:** CSS Grid, Flexbox, custom breakpoints
